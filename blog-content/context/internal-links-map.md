## 내부 링크 전략

### 블로그 → 전자책 (주요 전환 경로)
- 후각 관련 글 → `/ebook/olfactory-training` 또는 `/ebook/olfactory-development`
- 수면 관련 글 → `/ebook/sensory-sleep`
- 부모 돌봄 글 → `/ebook/healing-parent`
- 키트/도구 관련 글 → `/ebook/olfactory-kit`

- 같은 키워드 클러스터의 글끼리 상호 링크
- pillar 글에서 cluster 글로, cluster 글에서 pillar 글로
