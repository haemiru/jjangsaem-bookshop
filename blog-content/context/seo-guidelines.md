# 짱샘의 치유책방 — SEO 가이드라인

## 타겟 검색엔진
- **Google Korea** (google.co.kr) — 1차 타겟
- **Naver** — 2차 타겟 (블로그/지식인 노출 고려)

## 핵심 키워드 클러스터

### 클러스터 1: 후각 훈련
- 후각 훈련, 후각 훈련 방법, 후각 발달, 후각 재활
- 후각 운동, 냄새 훈련, 후각 기능 회복
- 아이 후각 발달, 감각 훈련

### 클러스터 2: 수면 치유
- 아이 수면 훈련, 숙면 방법, 수면 개선
- 아기 잠투정, 수면 루틴, 수면 환경
- 아로마 수면, 향기 수면 치유

### 클러스터 3: 감각 치유
- 감각 통합 치료, 감각 처리 장애
- 발달장애 감각 훈련, 감각 자극 활동
- 오감 발달, 감각놀이

### 클러스터 4: 부모 자기돌봄
- 엄마 자기돌봄, 양육 스트레스 해소
- 부모 번아웃, 부모 치유
- 돌봄 피로, 양육 우울

### 클러스터 5: 아로마테라피
- 아로마테라피 효과, 에센셜 오일 활용법
- 라벤더 효능, 아로마 디퓨저
- 아이에게 안전한 아로마

## 경쟁 분석 대상
- 네이버 블로그 상위 노출 글
- 브런치 감각 치유 관련 글
- 맘카페 인기 글
- 관련 YouTube 콘텐츠

## Schema.org 마크업
- 블로그 글: `Article` 또는 `BlogPosting`
- FAQ 포함 시: `FAQPage`
- 방법 안내: `HowTo`
- 제품 소개: `Product`

## 한국어 SEO 특수 고려사항
- 조사(은/는/이/가) 변형 키워드 고려
- 띄어쓰기 차이 고려 ("후각훈련" vs "후각 훈련")
- 한글+영어 혼용 키워드 ("아로마테라피" = "아로마 테라피")
- 네이버 검색 최적화: 제목에 핵심 키워드, 본문 초반 키워드 배치
- 모바일 최적화 필수 (한국 모바일 검색 비율 80%+)

## 콘텐츠 발행 전략
- 주 1-2회 블로그 발행
- 전자책 내용 기반 파생 콘텐츠 (챕터 요약, 실습 팁)
- 시즌별 키워드 공략 (봄: 알레르기+후각, 겨울: 수면 루틴)
- 긴글(pillar content) + 짧은글(cluster content) 병행
